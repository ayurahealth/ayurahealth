## 2025-05-18 - Component Unnecessary Rerenders Optimization\n**Learning:** Passing inline anonymous functions like `onSelectSource={() => {}}` as props to deep component trees (like `ChatInterface` -> `ChatMessagesPanel` -> `MessageItem`) causes unnecessary re-renders of the entire message list on every parent update, which is especially problematic in streaming chat applications.\n**Action:** Memoize empty handler functions using `useCallback` and wrap individual list item components in `React.memo` to prevent full-list re-renders on streaming updates or keystrokes.
